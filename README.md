## 📋 Sobre Mi

QA Automatizador Sr con experiencia en testing manual y automatizacion de pruebas. Especializado en **pruebas de APIs y aplicaciones web** usando **Playwright** como framework principal. Experiencia en **CI/CD con GitHub Actions** y **Docker** para entornos reproducibles.

Actualmente trabajo como **QA Automation Sr**, disenando e implementando estrategias de testing end-to-end, automatizando flujos criticos de negocio y colaborando con equipos multidisciplinarios para garantizar calidad en cada release.

## 💼 Experiencia Profesional

### **QA Automation Sr** — Empresa Actual (2023 - Presente)
Automatizacion de pruebas web y APIs con Playwright. Implementacion de CI/CD con GitHub Actions y configuracion de entornos reproducibles con Docker. Diseno de frameworks con Page Object Model, fixtures y reportes Allure. Colaboracion con desarrolladores y producto para shift-left testing.

### **QA Automation Engineer** — Empresa Anterior (2021 - 2023)
Automatizacion de pruebas web y APIs. Implementacion de Page Object Model, fixtures, reportes Allure. Configuracion de pipelines de CI/CD y pruebas de regresion automatizadas.

### **QA Manual Tester** — Empresa Inicial (2019 - 2021)
Pruebas funcionales, de regresion y exploratorias. Diseno de casos de prueba basados en criterios de aceptacion. Reporte de bugs y colaboracion con el equipo de desarrollo.
